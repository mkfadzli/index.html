# Infographic Factsheets

Interactive HTML reference guides for telecom and data topics. Each page is a self-contained, dark-themed infographic built with Tailwind CSS.

## Pages

| File | Description |
|------|-------------|
| [index.html](index.html) | Landing page with links to all factsheets |
| [nr-factsheet.html](nr-factsheet.html) | **5G NR in a Nutshell** — Engineer's quick reference for 5G New Radio |
| [lte-factsheet.html](lte-factsheet.html) | **LTE in a Nutshell** — Engineer's quick reference for LTE |
| [sql-factsheet.html](sql-factsheet.html) | **SQL in a Nutshell** — Beginner edition for SQL fundamentals |
| [link-budget-factsheet.html](link-budget-factsheet.html) | **Link Budget in a Nutshell** — Engineer edition for RF path loss and margins |
| [python-programming-factsheet.html](python-programming-factsheet.html) | **Python Programming in a Nutshell** — Engineer edition for Python 3 fundamentals |
| [linux-factsheet.html](linux-factsheet.html) | **Linux in a Nutshell** — Engineer edition for kernel, VFS, and system administration |
| [bash-scripting-factsheet.html](bash-scripting-factsheet.html) | **Bash Scripting in a Nutshell** — Engineer edition for GNU Bash 5 and POSIX shell scripting |

## GitHub Pages

This repository is published with GitHub Pages. Visit the site at:

**https://mkfadzli.github.io/index.html/**

To enable or update Pages:

1. Go to **Settings → Pages** in the repository.
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
3. Choose the `main` branch and the `/ (root)` folder.
4. Save.
